This page tracks flows into/out of the largest 100 US Equity ETFs across SSGA, iShares and Invesco for S&P 500 constituents. The data pipeline runs on GCP (Google Cloud Platform).

These numbers give an idea of the movement of capital invested in ETFs, and hence give colour on macroeconomic trends.

## How it Works

This project uses an ETF scraper to collect ETF holdings and a few scripts to run a data pipeline on Google Cloud Platform:

1. Scrape ETF holdings and save to GCS (cloud storage) — funds scraped are specified in a config file on GCS
2. Load the holdings files into BigQuery (GCP database)
3. Calculate flows for S&P 500 constituents using a SQL query
4. Format the results into HTML tables and output to GCS
5. Run a GitHub Actions workflow to pick up the results and publish them

Steps 1–4 run as a scheduled Google Cloud Run Job.
