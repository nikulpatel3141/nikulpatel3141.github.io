## General
My advice to learn a new programming language quickly:
1 - Have a project you want to build in that language
2 - Learn the syntax of the language
3 - Build the project

If you find step 1 hard, it's likely you'll have a bad time learning. If you have no use case for a language, do you really need to learn it?

To learn syntax quickly, I'd recommend you write your own "Hello, World!" script in that language. This will be a program whose logic you're familiar with and that is somewhat complex. For example, my "Hello, World!" script is a program to generate the Mandelbrot set and save the result - this covers many basic constructs (assignment, arithmetic, for, while, ...).

If you can try to build something with visual feedback. It is highly rewarding to actually see the outcome of your work; you will become addicted and you will be naturally driven to finish the project. (Maybe that's why I've enjoyed building this website so much...)
